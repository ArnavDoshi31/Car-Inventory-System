The Advanced Car Inventory & Sales Management System is a comprehensive, full-stack web application designed to streamline the operations of a modern automotive dealership. Developed entirely from scratch without the use of heavy backend frameworks like Spring Boot or Node.js, this project demonstrates a deep, fundamental understanding of network protocols, server architecture, and file-based persistence.
The backend is powered by standard Vanilla Java, utilizing the built-in com.sun.net.httpserver.HttpsServer to create a lightweight, multithreaded REST API. The persistence layer operates through a custom-built CSV repository, teaching core concepts of File I/O and data serialization. On the client side, the application features a stunning Single Page Application (SPA) built with Vanilla HTML, CSS, and JavaScript. 
It employs modern UI/UX principles, specifically the "Glassmorphism" aesthetic with animated gradients, and integrates Chart.js for real-time, cumulative sales and profit analytics. This report outlines the architecture, implementation, and successful deployment of a dependency-free, highly responsive dealership management platform


In the fast-paced automotive retail industry, dealerships require robust software to track vehicle inventory, manage customer relationships, log employee data, and analyze sales performance. Many existing solutions are either overly complex, requiring massive server resources and steep learning curves, or too simplistic, relying on manual spreadsheet entry prone to human error. The primary objective of this project was to bridge this gap by developing a custom, lightweight inventory system that is both technically rigorous and visually captivating.
By deliberately avoiding heavy frameworks, this project serves as a profound academic exercise in understanding the core mechanics of web development. From manually parsing JSON strings and decoding Base64 image uploads to writing custom HTTP routing logic, the system relies purely on the foundational capabilities of Java and standard web technologies. The resulting application not only meets all functional requirements for Dealership Management (CRUD operations across multiple entities) but also provides an incredibly sleek, modern user experience that rivals commercial software.
